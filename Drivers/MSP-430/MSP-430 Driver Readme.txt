If you are using a MSP-430, please download the appropriate drivers
    i.	If you are using a Linux Computer:
        1.	Please download the TI udev rules here. This is required to run the Energia IDE not as root. 
        2.	Execute the following command: sudo mv /71-ti-permissions.rules /etc/udev/rules.d/
        3.	No drivers required for installation. 
    ii.	If you are using a Mac Computer:
        1.	Please download the drivers here. 
    iii.	If you are using a Windows computer:
        1.	Please download the drivers here.
            a.	These drivers are not signed and may require you to disable the signed driver feature in windows. Please click here for a guide on how to do so.
